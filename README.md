A simple Python-based local LLM chat application using Ollama.

This project demonstrates how to interact with a locally running large language model (LLM) via HTTP API, maintain conversation history, and simulate memory across multiple turns.

Features:
- Connects to a local Ollama server
- Supports multi-turn conversations
- Maintains chat history (context)
- Uses system prompts to control behavior

Tech Stack:
- Python
- Requests
- Ollama (local LLM runtime)
